# LogKits (Система китов с ReallyWorld)

Плагин , который позволяет создавать и управлять наборами предметов с системой кулдаунов и разрешений, также в этом плагине реализован просмотр набора

## Возможности

-  Создание и управление китами
-  Система кулдаунов для каждого кита
-  Система разрешений
-  GUI для предварительного просмотра китов
-  Поддержка брони и предметов в оффхенде
-  Настраиваемые сообщения
-  Настраиваемый интерфейс
-  Автоматическое сохранение данных

## Установка

1. Скачайте файл плагина (.jar)
2. Поместите его в папку `plugins` вашего сервера
3. Перезапустите сервер
4. Настройте плагин через конфигурационные файлы
